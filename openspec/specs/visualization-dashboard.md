# 互動式視覺化儀表板

## 狀態

draft

## 概述

提供網頁介面讓使用者以互動式圖表探索台灣公開資料，支援多種圖表類型與資料篩選功能。

## 需求

### 圖表類型

- 折線圖（時間序列趨勢）
- 長條圖（類別比較）
- 圓餅圖（佔比分佈）
- 地圖視覺化（台灣縣市資料分佈）

### 互動功能

- 滑鼠懸停顯示資料細節（tooltip）
- 點擊圖表元素進行篩選/鑽取
- 時間範圍選擇器
- 資料集切換選單

### 頁面架構

- 首頁：資料集總覽與精選視覺化
- 資料集頁面：單一資料集的詳細視覺化
- 支援 RWD（響應式設計）

## 非目標

- 不提供使用者自訂圖表功能（初期為預設視覺化）
- 不實作使用者帳號系統
- 不支援資料匯出功能

## 技術方案

- 使用 D3.js 繪製互動式圖表
- 純前端 HTML/CSS/JavaScript（由 FastAPI 提供靜態檔案）
- 透過 Fetch API 呼叫後端 REST API 取得資料
